🍇 Guia de Instalação

Açaí Seguro - Sistema de Gestão de Qualidade Sanitária

⚠️ Erro "psql não reconhecido"?

Este guia mostra como configurar o sistema SEM precisar do comando psql!

1 Instalar Dependências

Abra o PowerShell na pasta do projeto e execute:

cd c:\projetos_web\vig_bat_acai\server
npm install

Aguarde a instalação terminar (pode demorar alguns minutos).

⚠️ Erro "npm não encontrado"?
Instale o Node.js primeiro: nodejs.org

2 Configurar Banco de Dados

Abra o pgAdmin (já vem com o PostgreSQL):

  1. Clique com botão direito em "Databases"
  2. Selecione "Create" → "Database..."
  3. Nome: acaiseguro
  4. Clique em "Save"

Executar o Schema:

  1. Clique no banco acaiseguro (menu lateral)
  2. Menu superior: "Tools" → "Query Tool"
  3. Clique no ícone 📁 "Abrir arquivo" (ou Ctrl+O)
  4. Navegue até: c:\projetos_web\vig_bat_acai\sql\schema.sql
  5. Clique em "Abrir"
  6. Clique no ícone ▶️ "Execute" (ou F5)
✅ Verifique:
No menu lateral: acaiseguro → Schemas → public → Tables
Deve ter 5 tabelas: batedores, checklists, calculos_cloracao, selos, requisitos_selos
⚠️ Não tem PostgreSQL?
Baixe em: postgresql.org
Durante a instalação, anote a senha do usuário postgres!

3 Configurar Credenciais

Na pasta: c:\projetos_web\vig_bat_acai\server\

  1. Copie o arquivo .env.example
  2. Renomeie a cópia para .env
  3. Abra o arquivo .env com Bloco de Notas
  4. Altere a linha:
DB_PASSWORD=SUA_SENHA_AQUI

Coloque a senha que você definiu na instalação do PostgreSQL.

⚠️ IMPORTANTE: Salve o arquivo após alterar!

4 Iniciar o Servidor

Opção A - Com o script:

cd c:\projetos_web\vig_bat_acai
.\start-server.bat

Opção B - Manualmente:

cd c:\projetos_web\vig_bat_acai\server
npm run dev
✅ Servidor funcionando quando aparecer: 
╔═══════════════════════════════════════╗
║   🍇 API AÇAÍ SEGURO 🍇              ║
║   Servidor rodando em:                ║
║   http://localhost:3000              ║
╚═══════════════════════════════════════╝

5 Abrir o Aplicativo

Clique duas vezes no arquivo:

c:\projetos_web\vig_bat_acai\index.html

O aplicativo abrirá no seu navegador!

📋 Checklist de Instalação

🧪 Testar se Está Funcionando

Teste 1 - API:

Abra no navegador: http://localhost:3000

Deve mostrar uma mensagem JSON.


Teste 2 - Cadastro:

  1. Abra o aplicativo (index.html)
  2. Vá em "Cadastro"
  3. Preencha o formulário
  4. Clique em "Salvar Cadastro"
  5. Deve aparecer mensagem de sucesso

Teste 3 - Banco de Dados:

No pgAdmin, execute:

SELECT * FROM batedores;

Deve mostrar o cadastro que você criou!

🐛 Problemas Comuns

❌ "Port 3000 already in use"
Solução: Abra server\.env e mude PORT=3001
❌ "Erro ao conectar no banco"
Solução: Verifique a senha no arquivo .env
❌ "Cannot find module"
Solução: Execute npm install novamente
❌ Frontend não salva dados
Solução: Verifique se o servidor está rodando em http://localhost:3000
